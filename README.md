# university-login-portal
A python based university portal where students and faculty can securely  access the academic information.
A desktop application built using Python (Tkinter) and SQLite that provides secure login access for students and faculty.
Students can view personal details, courses, and timetables, while faculty members can access student information and manage data efficiently.
